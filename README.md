## Integrated Items

* Jenkins LTS 2.46.3
* Jenkins Plugins: see plugins.txt


## Usage

To run a nagios service with default config, use the command below:

    $ docker run -d -p 8080:8080 -p 50000:50000 guessi/docker-jenkins
